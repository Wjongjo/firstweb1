HTML

Hypertext Marktup Language (HTML) is the standard markup language for creating web pages and web applicatings. WEBbrowsers receive HTML documents from a web server of from local storage and render them into multimedia web pages. HTML describes the structure of a web page semantically and originally included cues for the appearance of the document.

HTML elements are the building blocks of HTML pages. with HTML constructs, images and other objects, such as interactive forms, may be embedded into the rendered page.
